¿Listo para desafiar a la pared con sólo tu cuerpo?

Vision Wall es un juego de acción que usa visión por computador para traducir tus poses en movimiento real. Detecta tu 
postura en tiempo real y genera retos físicos.

Funciona con seguimiento de pose, y a través del motor de colisiones de godot, evaluamos si se ha chocado o ha pasado por
el hueco de la pared. Este juego lo hemos enfocado además como un party game, focalizandonos en la diversión.

Jugar es sencillo: calibras tu cámara, y pasa por el muro, así de facil. Asegurate de tener espacio 
suficiente antes de jugar, noqueremos accidentes. 

Para el desarrollo de este software hemos utilizado principalmente godot y python ya que ambos tienen soporte para 
websockets y nos permitia utilizar mediapipe y así posibilitar el desarrollo del videojuego

además hemos utilizado vs code para la gestión del proyecto, blender para modelaje 3d, kdenlive para la edición vídeo,
krita para los dibujos, gimp para la manipulación de imágenes, audacity para la grabación de audio, obs studio para
la captura de video y lmms para la creación de música.

Vision Wall — una aplicación práctica de visión por computador. Desarrollado por Nicolás Rey Alonso y Wafa Azdad Triki. 
Pruébalo y mueve tu cuerpo para ganar.
